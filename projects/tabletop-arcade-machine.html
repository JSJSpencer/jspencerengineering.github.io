TABLETOP ARCADE MACHINE

Custom-built retro gaming system with woodworking and electronics integration

Project Overview

Designed and assembled a tabletop arcade machine using Fusion 360 and Adobe Illustrator, creating and assembling all of the wood and acrylic components using CNC routers, laser cutters, and Fusion 360 Manufacture.

Installed a Raspberry Pi 3B+ to run the Linux-based operating system, RetroPie, and attached peripherals such as buttons, speakers, joysticks, and screens, enabling emulation of Gameboy, Nintendo DS, arcade-style games, and various others (200+ games) and supporting up to 4 players through various controller types.
