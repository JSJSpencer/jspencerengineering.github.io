STEALTH PLANES MUSEUM EXHIBIT

Interactive spy technology exhibit with real-time radar detection system

Project Overview

Designed and prototyped an interactive museum exhibit featuring spy technology using MATLAB and Arduino. The system detects the placement of planes on a board and overlays the coordinates onto a radar display, using a custom circuit board and code.

The exhibit accurately determines the plane's placement and identifies it as friendly or enemy aircraft through sophisticated detection algorithms. This creates an engaging educational experience that demonstrates radar technology and stealth aircraft concepts to museum visitors.

The project combines hardware engineering with software development to create a fully functional interactive display that responds in real-time to visitor inputs.
