MINI ROBO ARM

Precision robotic arm with servo motor control and programmable automation

Project Overview

Designed and built a compact robotic arm using servo motors and an Arduino microcontroller, featuring multiple degrees of freedom for precise object manipulation and automated task execution.

The system features a hotswappable end effector design with automatic tool detection through a pogo pin connection, allowing seamless switching between different tools and attachments while maintaining electrical continuity for sensor feedback and control.
